üé≤ Exercise 8: Complete Dice Generator Project

Lab Exercise 8: Dice Project

You have been hired by devsInc to create a landing page for their new project. They want to support people playing tabletop games from home and require a Dice Generator.

Features:

  • The user should be able to choose different dice, such as a D6 or a D10 (number of faces)
  • The user should be able to roll their chosen dice to see the random outcome

Requirements:

  • Define on paper, all the user stories (functionalities) you will need for this page
  • Make a simple storyboard of the user using the system
  • Create a git repository for the project
  • Clone the project locally
  • Work on the project creating 2 branches, one for the UI, and one for rolling the dice, committing and merging when completed
  • Create the roll dice function by passing an argument, in order to reutilize the same function multiple times
  • Write unit tests for the roll dice function
  • When complete, push the content to your git repository

üìã Project Planning & User Stories

üéØ User Stories Defined:

  1. Select from a list of dice types
  2. Click a "Roll" button to roll the selected die
  3. See the result displayed on screen
  4. Roll multiple times without reloading
  5. Responsive UI for desktop and mobile
  6. Extension: Track roll history
  7. Extension: Roll multiple dice at once

üé¨ Storyboard Process:

Frame 1: Landing page with title, dropdown, roll button
Frame 2: User selects dice type (e.g., D10)
Frame 3: User clicks roll ‚Üí result appears
Frame 4: User can repeat without reload

// Clean onclick approach - no event listeners function handleRoll() { const faces = parseInt(document.getElementById("dice-select").value, 10); const resultDiv = document.getElementById("result"); resultDiv.classList.add("rolling"); setTimeout(() => { const result = rollDice(faces); resultDiv.textContent = `${result}`; addToHistory(faces, result); // Compact 6-item history }, 600); } // Global function availability window.handleRoll = handleRoll;
